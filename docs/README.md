## Documentation

This folder contains documentation for the Smart Waste Management System.

- **[smart-waste-project-governance.md](smart-waste-project-governance.md)** – **Governance of architecture, security, and agents.** Single source of truth for project scope, technical decisions, roles, and implementation phases.
- **[diagrams-and-blocks-for-word.md](diagrams-and-blocks-for-word.md)** – **Blocks, tables, and diagram code for Word.** Copy-paste blocks and Mermaid diagram code to insert into project documentation in Microsoft Word (use [mermaid.live](https://mermaid.live) to export diagrams as PNG/SVG).
- `requirements.md` – detailed functional and non-functional requirements (to be expanded from `Agents.md`).
- `architecture.md` – high-level architecture, component diagrams, and data flow.
- `api-spec.md` – contracts for backend REST/WebSocket APIs.
